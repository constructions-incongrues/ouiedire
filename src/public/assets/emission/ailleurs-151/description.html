Suivez les murmures et les échos, laissez les guitares vous guider, car oui, elles savent parfaitement où elles vont.
Pensiez-vous que le temps s'était évaporé ? Toutes ces couleurs vous ont-elles fait tourner la tête ?
N'ayez crainte, car dans ce doux brouillard, les regards trouveront toujours le moyen de se croiser.

Peinture de Barbara Burke - Passing through pink ( 1925 )
